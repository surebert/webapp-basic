Example Surebert Framework Project

This page is ugly on purpose. You should get rid of it.

  1. Make sure you have a copy of the surebert framework on your server. It is usually found in /var/www/sbf For more info see the surebert framework deployment guide.

    If you have deployed the framework elsewhere or wish to specify a certain tag, set the SUREBERT_FRAMEWORK_SB_PATH and SUREBERT_TOOLKIT_PATH constants in /private/config/definitions.php to where sbf is deployed at. 

    define("SUREBERT_FRAMEWORK_SB_PATH", "/some/other/path/sb/tags/1.65");
OR
define("SUREBERT_FRAMEWORK_SB_PATH", "/var/www/sbf/sb/tags/1.65");

  2. In order to get rid of this page, uncomment the following line in /public/.htaccess 

    #RewriteRule .* index.html

After that, your site should now be ready and you can refresh this page! Read up on the surebert framework and toolkit @ surebert.com and on the surebert wiki
